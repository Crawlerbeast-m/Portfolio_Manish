Netflix-Clone

This project is a Netflix-inspired web application that dynamically fetches and displays movie and TV show data using The Movie Database (TMDb) API. It features categories like "Netflix Originals," "Top Rated," "Action Movies," "Comedy Movies," "Horror Movies," "Romance Movies," and "Documentaries." The app uses JavaScript to fetch data, dynamically create DOM elements, and render movie posters and banners. It includes a truncate function to limit text descriptions and ensures a responsive, visually appealing layout. The project demonstrates skills in API integration, DOM manipulation, and asynchronous programming with fetch and Promises.
